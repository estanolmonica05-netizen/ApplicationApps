🎥 Live Object Detection

An interactive real-time object detection web application built with **Streamlit, YOLOv8, OpenCV, and WebRTC**.  
This project uses AI-powered computer vision to detect objects live through your webcam with real-time alerts and object counting.


📖 Features

📷 **Real-Time Detection** – Detect objects instantly using your webcam.
🤖 **YOLOv8 AI Integration** – Fast and accurate object recognition.
🚨 **Custom Alert System** – Get alerts for selected objects.
📦 **Object Counting** – Displays the total number of detected objects.
💾 **Frame Saving** – Automatically save detected frames.
🎨 **Modern UI Design** – Clean Streamlit interface with custom styling.
⚡ **Live Video Streaming** – Powered by WebRTC for smooth performance.

 🛠️ Technologies Used

 **Python** – Main programming language
 **Streamlit** – Web application framework
**Streamlit WebRTC** – Real-time video streaming
**Ultralytics YOLOv8** – AI object detection model
**OpenCV** – Image processing
**AV** – Video frame handling
